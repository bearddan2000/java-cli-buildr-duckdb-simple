# java-cli-buildr-duckdb-simple

## Description
Reads from csv and json giving field names.

## Tech stack
- java
- buildr
  - duckdb

## Docker stack
- vanto/apache-buildr:latest-jruby-jdk8

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
